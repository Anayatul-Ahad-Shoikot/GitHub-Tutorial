Steps:

1

Open your project folder in VS Code (or any terminal).

2

git init Initializes Git in your project folder and creates a hidden .git directory. This tells Git to start tracking changes in this folder.

3

Go to your GitHub account and create a new repository. Do not add a README, .gitignore, or license â€” keep it empty to avoid merge conflicts.

4

git remote add origin <repo_link> Links your local project with the GitHub repository (only needed once).

5

git add . Stages all changes so Git knows what to include in the next commit. For a single file: git add filename

6

git commit -m "Your message" Saves your staged changes into Git history with a descriptive message.

7

git push -u origin main Uploads your commits to GitHub's main branch. The -u flag sets origin/main as the default branch for future pushes.
